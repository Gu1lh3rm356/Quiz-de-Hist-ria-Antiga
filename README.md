# Questionário Histórico
##Tente acertar todas as questões. Se acertar todas,vc é legáu
Este é um jogo interativo de perguntas e respostas com 25 questões sobre História Geral e do Brasil. O projeto possui um sistema de dicas e explicações detalhadas para cada alternativa, servindo como uma ferramenta dinâmica de aprendizado. Originalmente estruturado em Python para rodar via terminal, ele valida as entradas do usuário e contabiliza a pontuação final. É um código modular livre para modificações, ideal para estudantes e entusiastas de programação e história.
